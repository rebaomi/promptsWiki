# promptsWiki

This repository now keeps separate English and Chinese Markdown versions. The original files remain as lightweight navigation pages.

## Folder Structure

- `advanced/`: general advanced prompts
- `ecommerce/`: e-commerce product and campaign prompts
- `portraits/`: portrait prompts
- `anime/`: anime prompts
- `movie-posters/`: movie poster prompts
- `ui/`: website and app UI design prompts
