# ecommerce PROMPT-TEMPLATES

这个文件只放电商和商业产品图的骨架模板。

## 白底主图模板

```text
Create a clean e-commerce hero image of [product] on a pure white background. [view angle], soft even studio lighting, subtle grounding shadow, crisp edges, accurate proportions, premium material detail, commercial catalog quality, no text, no extra props
```

```text
生成一张电商白底主图：主体是[产品]，放在纯白背景上，[角度/视角]，柔和均匀棚拍光线，底部保留轻微自然阴影，边缘清晰，比例准确，材质真实高级，整体像商业目录图，不要文字，不要多余道具。
```

## 高端广告图模板

```text
Create a luxury advertising image of [product] placed on [stone/pedestal/surface]. [palette], [lighting], [brand tone], sharp focus on material texture, minimal but elegant composition, no extra props, no text artifacts
```

```text
生成一张高端广告图：一件[产品]放在[台座/表面]上，整体为[配色]，[光线方式]，风格偏[品牌气质]，突出材质、边缘高光和表面细节，构图简洁高级，避免多余道具，避免错误文字。
```

## 生活方式场景图模板

```text
Generate a lifestyle product image of [product] in [environment]. [time of day] light, [props kept minimal], [palette], editorial commercial photography style, clean composition, no clutter
```

## 详情页特写模板

```text
Create a macro detail shot of [product], focusing on [material/stitching/hardware/texture]. Controlled studio light, sharp focus, tactile realism, luxury e-commerce detail aesthetic, no clutter
```

## 社媒投放图模板

```text
Create a square or vertical social ad visual for [product], [background style], centered composition, clear negative space for future copy, premium campaign aesthetic, no text, no clutter
```
