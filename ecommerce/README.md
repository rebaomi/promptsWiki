## 适用场景

适合：
- 白底主图
- 高端广告图
- 生活方式场景图
- 详情页特写
- 社媒投放图

## 使用建议

1. 如果是商品图，先确定是白底还是场景图。
2. 如果是品牌广告，优先补材质、台座、打光和留白要求。
3. 如果要直接商用风格，建议叠加 `advanced` 里的质感和控制词。

## 推荐搭配

- 商品详情页: `ecommerce` + `advanced`
- 高端品牌广告: `ecommerce` + `movie-posters` 中的排版思路
- 社媒投放封面: `ecommerce` + `ui` 中的版式思路
