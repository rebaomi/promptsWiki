# ui PROMPT-TEMPLATES

这个文件只放网站、App、dashboard 和展示图类的骨架模板。

## 网站落地页模板

```text
Create a premium website landing page for [product/brand], desktop layout, [hero structure], [visual style], [palette], [typography tone], clear CTA, realistic UI hierarchy, polished product design, no clutter
```

```text
生成一张高质量网站落地页设计图：产品是[产品/品牌]，桌面端布局，[首屏结构]，[视觉风格]，[配色]，[字体气质]，CTA 清晰，信息层级明确，整体像成熟产品官网，避免杂乱。
```

## Dashboard 模板

```text
Generate a dashboard UI for [product/use case], left sidebar, top filters/search, modular cards, charts, KPIs, [palette], polished enterprise product design, readable spacing, no clutter
```

## 移动端首页模板

```text
Create a mobile app home screen for [app type], [main modules], bottom tab bar, [palette], rounded cards, clean modern product design, strong hierarchy, no unreadable labels
```

## 登录注册模板

```text
Generate a sign-up/login page for [product], [split or centered layout], clean auth card, subtle illustration or gradient, modern typography, polished startup UI quality, no alignment issues
```

## 作品展示模板

```text
Create a Dribbble/Behance-style presentation shot for [app/site], multiple screens or one hero mockup, clean studio background, premium lighting, coherent design system, polished portfolio presentation
```
