## 适用场景

适合：
- 网站落地页
- Dashboard / 数据后台
- 移动端 App 页面
- 引导页与登录注册
- Dribbble / Behance 展示图

## 使用建议

1. 先明确是网站还是移动端。
2. 再确定是产品首页、功能页、dashboard 还是展示图。
3. 如果想要更强品牌视觉，可以叠加 `advanced` 的质感词或 `movie-posters` 的视觉语言。

## 推荐搭配

- SaaS 官网: `ui` + `advanced`
- 品牌感产品页: `ui` + `movie-posters`
- 电商首页概念图: `ui` + `ecommerce`
