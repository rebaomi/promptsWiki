# promptsWiki

这个仓库现在把 Markdown 文档拆成了独立的英文版和中文版，原始文件保留为导航页。

## 目录结构

- `advanced/`: 高级通用提示词
- `ecommerce/`: 电商产品与广告提示词
- `portraits/`: 人物写真提示词
- `anime/`: 动漫与二次元提示词
- `movie-posters/`: 电影海报提示词
- `ui/`: 网站与 App UI 设计提示词
