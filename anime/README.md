## 适用场景

适合：
- 角色立绘
- 动漫头像与半身图
- 场景与背景图
- 海报感主视觉
- 治愈系与热血动作系

## 使用建议

1. 先选角色图还是场景图。
2. 再选偏治愈、校园、幻想、动作还是海报感。
3. 如果需要更强的宣传感，可以叠加 `movie-posters` 目录里的海报模板。

## 推荐搭配

- 动漫角色海报: `anime` + `movie-posters`
- 动漫场景氛围图: `anime` + `advanced`
- 动漫 App 封面或视觉展示: `anime` + `ui`
