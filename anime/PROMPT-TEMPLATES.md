# anime PROMPT-TEMPLATES

这个文件只放动漫、二次元、角色与场景类骨架模板。

## 角色立绘模板

```text
Create a clean anime character illustration of [character description], wearing [outfit], [pose], full-body, polished line art, soft cel shading, [background hint], no extra limbs, no clutter, no text
```

```text
生成一张日系动漫角色立绘：主体是[角色描述]，穿着[服装]，姿态为[姿态]，全身完整展示，线稿干净，赛璐璐上色，[背景提示]，避免多余肢体，避免杂乱，避免文字。
```

## 头像模板

```text
Generate an anime bust portrait of [character], [expression], [palette], clean background, refined linework, polished cel shading, modern anime style, no distortion
```

## 场景图模板

```text
Create an anime background scene of [place], [time/weather], [lighting], [color palette], cinematic composition, polished anime movie background quality, no clutter, no warped perspective
```

## 海报感主视觉模板

