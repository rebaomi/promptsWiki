## 适用场景

适合：
- 高级时尚写真
- 自然生活写真
- 少女感与氛围写真
- 职业与纪实人像
- 情侣写真

## 使用建议

1. 先明确是时尚、生活化还是职业形象。
2. 再补环境、光线、镜头感和情绪描述。
3. 如果需要更强风格感，可以叠加 `movie-posters` 的情绪视觉语言。

## 推荐搭配

- 杂志感肖像: `portraits` + `advanced`
- 电影感人像: `portraits` + `movie-posters`
- 社媒头像或封面: `portraits` + `ui`
