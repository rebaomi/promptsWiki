# advanced PROMPT-TEMPLATES

这个文件只放可复制的通用骨架模板，适合作为所有专题的起手 prompt。

## 通用高级模板

```text
[主体] in [场景/环境], [构图方式], [光线氛围], [视觉风格], [材质与细节], [主色调], [镜头语言/视角], highly detailed, clean composition, strong focal point, professional quality, coherent spatial structure, no clutter, no distortion
```

```text
生成一张[风格/用途]图片：主体是[主体]，位于[场景]中，采用[构图]，整体为[时间/天气/光线]氛围，风格偏[风格方向]，强调[材质/纹理/表面细节]，主色调为[颜色方案]，镜头视角为[平视/俯视/低机位/特写]，画面干净，细节丰富，空间关系清晰，避免杂乱，避免畸形，避免多余物体。
```

## 摄影感模板

```text
[subject], in [location], [camera framing], [lighting], [lens], [depth of field], [mood], editorial photography, realistic textures, natural posture, clean background, premium quality
```

## 海报感模板

```text
[main subject], vertical poster composition, strong silhouette, dramatic lighting, [color palette], cinematic atmosphere, elegant negative space for title, premium key art quality, no clutter, no text distortion
```

## 产品图模板

```text
[product name and material], placed on [surface/pedestal], [composition], [studio lighting], [brand tone], [palette], sharp focus, premium material rendering, minimal background, commercial quality
```

## 插画模板

```text
[subject], [scene], [action], [illustration style], [linework/brushwork], [color palette], clear focal point, polished illustration, strong silhouette, no clutter
```

## UI 视觉模板

```text
[page or screen type] for [product/app/brand], [layout structure], [visual style], [palette], [typography tone], clear information hierarchy, realistic interface spacing, polished product design, no clutter, no unreadable labels
```
