## 内容范围

适合：
- 高级 prompt 基础结构
- 通用构图、光线、风格、材质、色彩控制
- 多类型跨场景 prompt 参考
- 中英文可复用模板

## 使用建议

1. 先从这里选一个基础结构。
2. 再到垂直专题目录里挑选具体场景模板。
3. 如果需要做更强风格化，再叠加电影海报或 UI 等专题方向。

## 推荐搭配

- 通用起手: `advanced` + 任一专题目录
- 高级出图: `advanced` + `movie-posters` / `ui`
- 产品商业图: `advanced` + `ecommerce`
