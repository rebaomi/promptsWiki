# GPT Image 2.0 电影海报字体排版专用提示词

这份文件聚焦电影海报里的片名设计、标语位置、演员名排布、演职员区块、标题层级和整体排版气质。

## 大标题排版

```text
生成一张院线电影海报，重点突出超大号居中片名。片名采用优雅的窄体 serif 字体，放在画面下三分之一位置，字距克制、层级明确、可读性强，整体具有高级电影感。顶部加入一句简短标语，底部加入克制的演职员信息区块，画面构图需要为专业排版预留足够空间，整体像正式上映海报。
```

```text
生成一张商业大片海报版式：使用非常醒目的粗体 sans-serif 片名横跨画面大部分宽度，现代、巨大、清晰易读，并与画面光影自然融合但不变形。演员名以小型大写字置于画面上方，片名前有一句 teaser 式短句，片名下方有真实感演职员区块。整体像主流片厂官方 one-sheet 海报。
```

## 极简文艺片排版

```text
生成一张极简电影节风格海报：片名很小，放在左上角，采用精致 serif 字体，大面积留白，整体安静克制。中上区域加入一行轻量标语，底部只保留极少量演职员信息。整体像国际电影节、艺术院线或文学改编电影海报。
```

```text
生成一张 A24 风格排版海报：整体字很少，片名使用小号大写字母，副标题与说明信息非常克制，排版疏朗、有呼吸感。字距、边距和层级都要高级，不要廉价视觉特效，不要文字拥挤，整体像高端独立电影官方海报。
```

## 竖排片名实验

```text
生成一张高级电影海报，片名沿画面一侧竖向排布，整体像编辑风或收藏版海报。字体干净、对齐精准、依附海报网格系统，另一侧保留一句简短标语用于平衡，下方仅保留少量演职员信息。整体像电影节 teaser 或限量收藏版印刷海报。
```

## 双语标题与定制字标

```text
生成一张收藏版电影海报，采用双语标题系统：主标题使用英文，副标题使用中文，放在下方或一侧，层级清楚、排版克制、对齐高级。再加入一条轻量标语和少量演职员信息，整体仍需保持官方海报质感，不要拥挤。
```

```text
生成一张电影海报，片名看起来像为这部作品特别设计的定制字标，整体高级、独特、略带质感，但必须保持清晰可读。演员名、上映信息和演职员区块使用辅助字体，层级更低且排布整洁。整体像真实片厂发布的主海报。
```

## 标语优先排版

```text
生成一张 teaser 预告海报，第一眼先看到一句短而有记忆点的标语。标语放在海报上半部分，字体克制但非常有存在感，片名缩小放在下方，整体神秘、自信、电影感强，像高端预热宣传海报。
```

## 演职员区块与片尾信息

```text
生成一张具有真实片厂排版逻辑的电影海报：底部加入正式演职员 billing block、发行信息和片尾说明区。billing block 要小、密、真实，符合院线海报习惯，但仍保持可读。片名必须是主视觉，标语次一级，其余信息层级清晰，整体平衡且官方感强。
```

## 为排版预留空间的构图

```text
生成一张专门为电影海报字体排版而设计的海报构图：顶部预留干净区域放标语，下三分之一预留大标题位置，最底部预留 billing credits 区域。关键文字区域后面不要有过于复杂的图像细节，整体像专业美术指导做好的电影海报底图，可直接进入正式排版阶段。
```
