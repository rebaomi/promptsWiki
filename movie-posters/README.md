## 使用建议

1. 先选 `主海报模板` 或 `收藏海报模板` 作为主体结构。
2. 再从 `风格方向` 里补一层视觉语言。
3. 如果需要真实海报成品感，再从 `字体排版` 里补标题和 credits 相关约束。

## 推荐组合

- 商业大片: 主海报模板 + 诺兰式史诗 / 赛博朋克 + 大标题排版
- 情绪文艺片: 主海报模板 + 王家卫式 / A24 风格 + 极简文艺片排版
- 收藏版角色海报: 收藏海报模板 + 港风 / A24 风格 + 双语标题或极简收藏排版
